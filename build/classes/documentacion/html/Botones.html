Botones

La aplicación consta de cuatro botones:

  • Matriculados en DI: este botón genera un listado con todos los alumnos matriculados en el módulo de DI.
  • Gráfico por sectores: este botón genera un gráfico por sectores que muestra el número de alumnos que hay matriculados en cada módulo.
  • Gráfico de barras: este botón genera un gráfico de barras que muestra el número de alumnos que hay matriculados en cada módulo.
  • Salir: este botón termina la aplicación.
